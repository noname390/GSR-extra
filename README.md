# GSR-extra

# 简介
幻想乡铁道是假想的位于东方project幻想乡的铁路系统，使用一个名叫BVE Trainsim的铁路模拟软件来再现这个铁道系统。这个虚拟线路采用了日本铁路的风格，并且以现实中日本信越地区的铁道系统为原型制作。在经过了大约半年的开发之后，博丽神社到守矢神社间的线路已经完成。在这之后BVE线路的开发暂时休止了一段时间。同时参加了同人展会来宣传幻想乡铁道项目，并且设计了几款周边产品。<br>
<br>
这个仓库分享的就是一些幻想乡铁道的周边产品的设计，以及在同人展会上设置摊位的建议。旨在帮助有自行制作周边产品和以幻想乡铁道为主题参加展会的意愿的爱好者。

# 在开始之前
代码仓库（幻想乡铁道的BVE线路）：https://github.com/noname390/BVE-Gensokyo-Railway<br>
如果您是初次了解这个项目，建议先访问上面的代码仓库，了解一些幻想乡铁道的基本设定之后，再接触周边产品。<br>
同时您需要对东方project和日本铁道都有一定的了解，这可以帮助您理解幻想乡铁道会涉及的各种要素。<br>

# 周边产品
在文件夹“products”里存放了一些周边产品的设计，您可以根据下面的指示制作和使用它们。<br>
根据MIT许可协议，您也可以在标明原作者（札幌人形）的情况下出售这些产品。所获得的收入自理，无需强制向原作者（我）上交收入的部分或全部。对作品的改动也是允许的，只要不过于离谱。<br>
这里的很多设计工作使用了draw.io，您可以在网上搜索并下载它。
## 徽章
在“徽章”文件夹内，同时提供了drawio源文件和图片。您可以在网络上寻找定制徽章的商家制作。我在上一次定制时使用的规格是44mm直径，材料为磨砂。您可以沿用这个规格或者自行调整它。<br>
注：徽章也被叫做“吧唧”，这两个词基本指同一种东西。
## 时刻表
目前的时刻表为供驾驶员使用的行车用时刻表。源文件是在“徽章”文件夹内的excel表格，在打印时应该使用300g以上厚度的A4卡纸。如果需要出售它，应该加上一个A4纸大小(21cm*29.7cm)或者相似大小的密封袋。
## 纸模
目前列车纸模有3种样式，分别为E127系、E129系、隙间列车，分别放在三个文件夹中。其中“final”文件夹内为最终的成品，可以直接打印，打印时需要用300g或以上厚度的卡纸以及彩色打印机。如果需要出售，将打印出的三张纸放入一个A4大小的密封袋内。<br>
注意：出售纸模时，为了与现实中的同型号列车进行区分，E127系、E129系应该写作GE127系和GE129系。<br>
## T恤
“T恤”文件夹内有效果图和衣物正面图案的原图，可以找定制衣物的商家制作。这个图案也可以印在其它地方，不限于T恤。

# 展会
这里的展会均指在中国大陆举办的东方project主题的同人展会，通常会以“THO”（TouHou Only）为简称。这类展会一般都会招收东方相关的同人社团参展。除此之外，部分东方主题的聚会，即“THP”（Touhou Party），也有可能设置同人社团摊位。<br>
您可以自行在任何相关展会上申请摊位，无需批准。但是为了避免引起误会，需要在摊位上明显位置标示”非官方摊位”字样<br>
展会上主要做的是两件事情：1. 展示幻想乡铁道的BVE线路，并让现场游客体验。2. 出售周边产品<br>
下面展示参加一场展会的各个步骤
## 找寻展会
### 获取展会信息
对于中国大陆范围内的展会活动，可以通过下面已知的两种途径获得：<br>
1. THBwiki<br>
在[THBwiki](https://thwiki.cc/)首页的右侧有一个日历，上面会标出活动的名称、日期、地点。在这个wiki中的“东方相关QQ群组列表“可以找到各个活动的官方群聊。<br>
2. QQ群机器人<br>
在常用的聊天软件“QQ”中的东方project相关群聊中可能会有机器人推送活动信息。常见的一个叫“赛博吟游机器诗人”，推送内容包括展会名称、时间、活动的群号。可以设置机器人为特别关注以防错过消息。
### 选择展会
可以通过个人的很多因素选择参加的展会，比如空闲时间、距离居住地的远近。此外应当优先考虑亚文化发达地区。根据2022年的东方人气投票估计，在32个省级行政区（不含港澳台）中，东方众分布密度最大的十个地区分别是上海、北京、天津、浙江、福建、广东、江苏、湖北、重庆、广西。（此处为自行计算所得，仅供参考）这些地区的铁道文化也较为发达，如有展会在这些地区举办，可以优先考虑。
## 申请摊位
一般通过展会的社团摊主群或邮件的形式申请摊位，这两种渠道一般会写在活动的官方群内。不同展会在此处会有一定差别，注意展会规模越大可能对作品要求越严格（比如要求必须有同人志等作品）。主办方一般会对摊位收取一定费用，但也有摊位免费的情况。在申请摊位时，应当使用统一的社团名”BVE幻想乡铁道“和"promotion"文件夹中的logo.png作为标志。<br>

## 准备摊位的物品+运送物料
### 准备摊位物品
摊位上的物品主要分为三类：出售的产品、展示模拟线路的设备、各种看板和标语。
#### 出售的产品
应该在开展前大约2个星期或更早就订购或制作好。并且将每一种类别清点好数量，最好能用电子表格记录。定价不宜过高，对于纸质的产品应在10元人民币以内。
#### 展示模拟线路的设备
BVE线路是幻想乡铁道项目的核心，因此展示它的设备是十分重要的。以我参加的上一场展会为例，使用的设备有：投影机、幕布、电脑主机、鼠标、键盘、电车GO手柄、耳机、各种电源线和数据线。<br>
也可以使用一个显示器（最好是24寸以上的）来代替投影机和幕布，可以使整套设备轻便很多。<br><br>
对于电脑主机，建议挑选便于携带的样式（如itx或更小的微型主机）或者使用笔记本电脑。为了使模拟线路可以流畅运行，电脑应该配有性能高于GTX 1050的GPU，以及8GB以上的RAM，对于CPU则无特别要求，CPU-Z高于1000分的均可用。<br><br>
对于手柄，可以在网上搜索”电车GO控制器 switch”寻找。电车GO版本的可以在淘宝买到，价格在1000元或以上。也有三鹰牌的手柄，更加接近真实的列车上使用的，但是比较难买到。如果方便出境，首选的购买地点是香港。可以先在谷歌上搜索“电车go switch手掣”，查询到店铺是否有货，然后现场购买。
<br>
必要时，可以增购一顶铁路制服的帽子和白色的礼仪手套。在体验列车驾驶的时候可以增强真实感。（可以取下原来的帽徽，换成幻想乡铁道的徽章）<br>
应该在出发之前在电脑上测试好幻想乡铁道的BVE线路，注意手柄要一同测试好。

#### 各种看板和标语
摊位上的各种标语可以给游客提供必要的信息。在signboard\print中的文件应该全部打印。<br>
注意：<br>
1. 把收款码换成自己的，标价牌上改成自己的定价。
2. 招牌由2部分组成，使用时首先将中间的城市名换成当前展会所在的城市，然后将2部分全部打印，用合适的方法拼接起来。
3. 注意更新项目贡献者名单，可以从这个readme开头的链接里找到。
4. 建议用厚卡纸打印
### 运送物料
展会用品的发货应该在开展前7天左右，如距离比较远则应当适当延长。
#### 打包
所有物品应该打包成两个纸箱，如果使用了投影仪，可能需要3个纸箱。其中场内出售的物品、看板与标语、驾驶台的手柄放在一个箱子内。显示器或者投影仪，以及耳机和各种电源线和数据线放在第二个箱子。第三个箱子用于在使用投影仪的情况下放置投影幕布和支架，应当是一个长条形的盒子。所有的箱子都应当注意放置减震材料，以免造成物品损坏。<br>
将几个箱子封好之后，注意在箱子外写上“展会名+社团名”，这里的社团名可以写“幻想乡铁道”、“GSR”等缩写，可以辨别即可。例如“札幌THO 幻想乡铁道”或“钏路THP GSR“。<br>
#### 发货
可以在快递平台上预约上门取件，或自行前往快递的门店。建议选择寄大重件。一般主办方会提供收件人信息，按上面的信息发货即可，同时订单号要保留好，以便双方知晓快递的状况。<br>
注：电脑主机建议在前往会场时随身携带，以免发生快递将电脑运坏的惨剧。
## 前往会场+布置摊位
### 前往会场
建议在开场的前一天到达会场所在城市，携带的物品参考平常旅行的样式即可，可以带上一套西装扮演日本铁路的职工，或者穿车万的cos服。注意带上电脑主机以及鼠标键盘、电源线、数据线、屏线。可以乘坐任意的自己喜欢的交通工具。<br>
如果是小型、中型的展会，官方可能会提供合宿，可以考虑参加。如果展会会场位于寒冷地区，十分建议住在会场所在酒店（小型展）或十分靠近会场的酒店（大型展）。<br>
### 布置摊位
一般布展的时间是开场前一天以及当天早上，如果会场的位置比居住的酒店房间低，可以先前往会场将电脑等物品放下，再前往客房。建议放置好所有展会上用不到的行李之后，再前去布展。<br>
#### 开场前一天
到达会场时，首先检查寄到的货物状况，先拿出所有电子设备测试好（注意打开BVE线路测试），然后清点剩下的物品。可以试着摆一下摊位上所有物品。如果使用投影机，可能需要将摊位的两个桌子前后叠放，才有足够的宽度放置演示所需的设备和其它物品（注意询问主办方是否同意这么做）。测试结束后将所有物品放回箱子中，避免丢失或被盗。<br>
建议当天晚上不要做影响第二天状态的事情，比如大吃大喝、熬夜、奖励自己（或者撅别人）等。<br>
#### 开场当天
建议在开展时间的两个半小时前或更早起床，有特殊衣服要迅速换好。吃完早饭就立即前往会场，将所有物品摆放为预定的样式，之后启动电子设备。这期间应该测试一遍BVE线路，确保没有问题。之后便可以等待开场了。
## 展会开场
上文已经说到，展会上需要做的是两件事：展示幻想乡铁道的BVE线路并让现场游客体验，以及出售周边产品。因此这里主要介绍的就是它们。
### BVE线路展示
注意在之前打印的标语中有一块牌子正反面写的是“演示中请稍候”和“开放体验”。开场的时候首先应该自己坐在位置上，亲自驾驶BVE中的列车，并让游客旁观。此时应该将牌子的“演示中请稍候”一面露出来。演示大约30到40分钟后（大概可以从博丽神社跑到守矢神社），便可以开放体验。此时将牌子换到“开放体验”的一面，等待游客前来体验。指导游客体验线路的全过程在“幻想乡铁道线路驾驶体验.docx”中有详细说明。此外，可以在展会前观看日本的铁道博物馆中模拟驾驶的视频，模仿铁道博物馆工作人员的指导方式。在游客体验结束后，可以引导其加入交流群或购买周边产品。<br>
**特别注意：这个驾驶体验不能收费，和BVE相关的很多素材均不能作商业用途**
### 周边产品出售
周边产品一般摆在桌子上展示，并且要使用之前打印的价格标签明码标价。建议提前打印物品清单，并使用正字计数法记录售出的物品数量。
### 其它
当需要离开摊位时，应当将“摊主离开，请稍等片刻“的牌子放置于桌子上。可以将电脑调至锁定状态（不过根据我的经验，THO/THP会场里的人一般比较有礼，没有遇到过乱动东西的情况）。<br>
对于中午吃饭，一般会场可以自带食品*，或者在会场附近的饭店用餐。如果是寒冷地区不便外出，则可以叫外卖。也有的会场会提供场内的食品供应（比如2024年的吉林THO）。<br>
一定要确保展会当天24小时都待在会场所在的城市，下文会讲到原因。<br><br>
**存在个别情况，请以实际为准。*
## 回送物料
### 收拾摊位
在展会结束时间之前的1~2小时，就可以开始收拾摊位了。首先关闭全部的电子设备，断开所有线缆的连接，将它们装回原来的快递箱中（电脑要整理好，在展会结束后放回随身行李中）。摊位上的看板以及剩余的周边产品装回原包装袋中，并放回箱子内，驾驶台的手柄也装回原来的包装，并放回原箱子中。
### 发货
在展会的当天下午就应该提前在快递平台上预约取件，当天取件的截止时间是特别早的（通常晚上7点之后就不行了），有大概率会遇到只能第二天取件的情况。因此一定要确保展会当天24小时都待在会场所在的城市。如果是自行在快递门店寄件，注意门店的揽收截止时间可能早于营业结束的时间。<br>
在发完货之后，如果没有别的事情，那么时间就比较宽裕了，可以在返程之前游览一些会场所在城市的地点（如果有fumo的话，带她们旅行也是一件很不错的事）<br>
